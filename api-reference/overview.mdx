## Resumen

La referencia interactiva de la API se genera automáticamente desde la especificación OpenAPI exportada de la colección de Postman de SIMA. Incluye los 137 endpoints agrupados por dominio, con schemas de request/response y un playground en vivo para probar llamadas directamente desde el browser.

<Info>
  La referencia de API estará disponible próximamente. Mientras tanto, usá la [colección de Postman](https://documenter.getpostman.com/view/12638281/2sA3Qy4Thh) como referencia autorizada de endpoints.
</Info>

## Headers requeridos

Cada solicitud debe incluir:

```
Authorization: Bearer {access_token}
X-SIMA-SYSTEM-ID: 8
Accept: application/json
Content-Type: application/json
```
