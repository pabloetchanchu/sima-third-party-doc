## Overview

The interactive API reference is generated automatically from the OpenAPI specification exported from the SIMA Postman collection. It includes all 137 endpoints grouped by domain, with request/response schemas and a live playground to test calls directly from the browser.

## Required headers

Every request must include:

```
Authorization: Bearer {access_token}
X-SIMA-SYSTEM-ID: 8
Accept: application/json
Content-Type: application/json
```
