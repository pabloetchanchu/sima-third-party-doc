## ¿Qué es SIMA?

[SIMA](https://sima.ag) — *Agricultura Conectada* — es una plataforma AgTech para gestionar toda la operación agrícola en un solo lugar: **planificación, ejecución y análisis en tiempo real**. Más de 10.000 usuarios la utilizan a diario en la región.

La plataforma digitaliza la operación de campo a través de ocho módulos integrados:

| Módulo SIMA | Qué cubre | Relación con la API de Terceros |
|---|---|---|
| **GIS** | Mapas, lotes y geometrías | `establishments`, `plots` |
| **Siembra** | Planificación y avance de siembra | `master_campaigns`, `campaigns`, `planting_progress` |
| **Monitoreo** | Recorridas, adversidades, NDVI, estados fenológico e hídrico | `scouting`, `adversity`, `hydric_states`, `phenological_states` |
| **Órdenes de trabajo** | Labores y aplicaciones de campo | `work_orders` + `/integration/api/v1/workOrders` |
| **Clima** | Registro pluviométrico y eventos por lote | `devices`, `device_events`, `climatic_factors`, `weather_anomaly` |
| **Analytics** | Reportes e indicadores de gestión | Consumo de datos vía endpoints de lectura |
| **Cosecha** | Avance y rendimiento de cosecha | `harvest_progress`, `progress_report` |
| **Post cosecha** | Silobolsas y almacenamiento en campo | `silobags`, `silobag_scoutings` |

La **API de Terceros** permite que ERPs, software agropecuario y plataformas externas lean y escriban datos en SIMA de forma segura — evitando la doble carga que genera tener la operación aislada en dos sistemas.

<Info>
  Más información sobre la plataforma en [sima.ag](https://sima.ag) o en la [landing de SIMA](https://sima-landing-246709467642.southamerica-east1.run.app/es/).

## ¿Para quién es esta API?

Esta API está diseñada para proveedores que integran con el ecosistema SIMA — el mismo que ya conecta productores con soluciones como Finnegans, Albor, Synagro, Acronex y otras plataformas de gestión:

- **Proveedores de ERP / software agropecuario** que necesitan sincronizar Órdenes de Trabajo, maestros y avances con SIMA
- **Apps de scouting y monitoreo** que complementan el módulo de Monitoreo Inteligente de SIMA
- **Plataformas de analítica** que consumen datos de campañas, clima y productividad
- **Cooperativas y agro-holdings** que centralizan el intercambio de datos entre múltiples productores
- **Proveedores de hardware IoT** (estaciones, pluviómetros) que alimentan el módulo de Clima

## Resumen técnico

- **Protocolo:** REST — verbos HTTP estándar (GET, POST, PUT, DELETE)
- **Formato:** JSON para todas las solicitudes y respuestas
- **Autenticación:** Basada en tokens via `POST /api/v2/login`
- **Versionado:** Endpoints en `v1`, `v2` y `v3` — consultá la [guía de versiones](/api-reference/versions) para saber cuál usar

1. Solicitá credenciales API a [soporte@sima.ag](mailto:soporte@sima.ag) o usá la [cuenta demo](/quickstart#cuenta-demo) para probar
2. Seguí el [Inicio Rápido](/quickstart) para hacer tu primera llamada a la API en menos de 5 minutos
3. Consultá la [Referencia de API](/api-reference/overview) con playground interactivo
4. Si integrás con un ERP, revisá el [Glosario SIMA ↔ ERP](/api-reference/sima-erp-glossary) antes de mapear Campañas, Cultivos y Ciclos
