## ¿Qué es SIMA?

SIMA es una plataforma de gestión agropecuaria que centraliza las operaciones de campo a través de establecimientos, campañas y cultivos. La API de Terceros permite que sistemas externos lean y escriban datos en SIMA de forma segura, habilitando la sincronización en tiempo real entre tu plataforma y las operaciones de tus clientes.

## ¿Para quién es esta API?

Esta API está diseñada para:

- **Proveedores de ERP / software agropecuario** que necesitan sincronizar Órdenes de Trabajo con SIMA
- **Apps de scouting y monitoreo** que registran observaciones de campo en tiempo real
- **Plataformas de analítica de datos** que consumen datos estructurados de campañas y productividad
- **Cooperativas y agro-holdings** que gestionan el intercambio de datos centralizado entre múltiples productores

## Resumen técnico

- **Protocolo:** REST — verbos HTTP estándar (GET, POST, PUT, DELETE)
- **Formato:** JSON para todas las solicitudes y respuestas
- **Autenticación:** Basada en tokens via `POST /api/v2/login`
- **Versionado:** Endpoints disponibles en `v1`, `v2` y `v3` — usá siempre la versión más reciente disponible
