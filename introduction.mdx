## What is SIMA?

SIMA is an agricultural management platform that centralizes field operations across farms, campaigns, and crops. The Third-Party API allows external systems to read and write SIMA data securely, enabling real-time synchronization between your platform and your clients' operations.

## Who is this API for?

This API is designed for:

- **ERP / agricultural software providers** that need to sync Work Orders with SIMA
- **Scouting and monitoring apps** that register field observations in real time
- **Data analytics platforms** consuming structured campaign and productivity data
- **Cooperatives and agro-holdings** managing centralized data exchange across multiple producers

## Technical overview

- **Protocol:** REST — standard HTTP verbs (GET, POST, PUT, DELETE)
- **Format:** JSON for all requests and responses
- **Authentication:** Token-based via `POST /api/v2/login`
- **Versioning:** Endpoints available in `v1`, `v2`, and `v3` — always use the latest available
